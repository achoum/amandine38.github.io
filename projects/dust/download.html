Installing Java

Dust requires for Java to be installed on your computer. If this is not the case, follow the instructions bellow:

Window
Download and install Java for Window.

Linux
ype sudo apt-get install default-jre in the console. Then run the game with java -jar dust_launcher.jar.
Alternatively, download and install Java for Linux.

Mac OS
Download and install Java for Mac OS.
